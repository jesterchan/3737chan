Para entendermos sobre a anatomia, as partes de um SQLI manual temos que saber oque é sqli... Sql injection é a injeção de instruções, querys em um banco de dados
por meio de alduteração de parametros ou outros meios de injeção. Essa é a explicação mais básica possivel, mais pra frente estáremos lançando tópicos com mais conteudos sobre
esse assunto.
Agora vamos falar sobre essa "anatomia", vamos usar como exemplo essa url: www.vuln.com/noticia.php?id=10 
A anatomia de um SQLI se baseia em 3 fatores: balanceamento,instrução e comentário

Vamos supor que a query criada pelo site seja SELECT * FROM noticia WHERE id='10' 
, iremos fazer uma injeção básica só para explicarmos os 3 fatores usando a clausula order

www.vuln.com/noticia.php?id=10' order by 1--+-

== PRIMEIRO FATOR "BALANCEAMENTO" ==

-> Como já vimos antes a query SELECT * FROM noticia WHERE id='10' oque vamos focar aqui é a parte do WHERE 
o balanceamento é a parte em que se dá o item ao id: id ='10', para balancearmos o sqli devemos terminar usando o caractere dado pelo site dentro do id
EXs: id='10' -> id=10' order by 1-- 
     id="10" -> id=10" order by 1-- 
	 id=(10) -> id=10) order by 1-- 
	 (id=(10)) -> id=10)) order by 1-- 
	 
	 www.vuln.com/noticia.php?id=10' order by 1--+-
	                              '-'
								BALANCEAMENTO
	 
== SEGUNDO FATOR "INSTRUÇÃO" ==

-> Instrução é a segunda query criada pelo injetor, tudo que vem depois do balanceamento...
EXs: www.vuln.com/noticia.php?id=10' order by 1--+-
                                 '-''--------'
					  BALANCEAMENTO  INSTRUÇÃO
					 
					 
== TERCEIRO FATOR "COMENTARIO" ==

-> Isso é oque fará com que tudo depois que vim depois da sua query não irá contar mais, por esse mótivo o balanceamento é muito importante para que haja tudo 
certinho =)
EXs:
--+-
--
;%00
#
%23

www.vuln.com/noticia.php?id=10' order by 1--+-
                           '-''--------' '---'
				BALANCEAMENTO INSTRUÇÃO	COMENTARIO	


== FINALIZAÇÃO ==

WWW.VULN.COM/NOTICIA.PHP?ID=10' order by 1--+-

' ->> BALANCEAMENTO
order by 1 ->> INSTRUÇÃO
--+- ->> COMENTARIO
